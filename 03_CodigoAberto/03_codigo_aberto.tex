\begin{frame}
	\frametitle{Definição de Software de Código Aberto}
	% Fonte: Open Source Initiative, http://www.opensource.org/docs/osd
	% Licença: Creative Commons Attribution 2.5 License
	Toda licença de distribuição de OSS deve respeitar os critérios:
	{\small 
    \begin{itemize}
\item[1] Redistribuição livre
% Esse item não impede que uma pessoa ou organização cobre uma taxa para  
%redistribuir o programa. Ela apenas garante o direito de redistribuir sem ter 
%que pagar nada para ninguém. 
\item[2] Código fonte incluído ou acessível
\item[3] Trabalhos derivados são permitidos\pause
\item[4] Integridade do código fonte do autor, restrições controladas à modificação (via \textit{patches})
%The license may restrict source-code from being distributed in modified form 
%only if the license allows the distribution of "patch files" with the source 
%code for the purpose of modifying the program at build time. The license must 
%explicitly permit distribution of software built from modified source code. 
%The license may require derived works to carry a different name or version 
%number from the original software. A licença Mozilla funciona assim.
\item[5] Sem discriminação contra pessoas ou grupos
\item[6] Sem discriminação contra áreas do conhecimento ou empreendimentos
%The license must not restrict anyone from making use of the program in a 
%specific field of endeavor. For example, it may not restrict the program from 
%being used in a business, or from being used for genetic research.
%A cláusula acima parece entrar em conflito com a GPL v3 na questão da 
%Tivoization.
%\item[7] Distribuição da licença
%The rights attached to the program must apply to all to whom the program is 
%redistributed without the need for execution of an additional license by those 
%parties.
% Isso significa que, ao distribuir software usando uma licença de código 
%aberto, não é possível agregar outras formas de licença que possam restringir 
%quem recebe o programa, por exemplo, não é permitido distribuir software de 
%código aberto para uma pessoa e, ao mesmo tempo, exigir que ela assine um NDA.
\item[8 e 9] A licença não pode ser específica para um produto ou distribuição 
e não deve restringir outros programas
%The rights attached to the program must not depend on the program's being part 
%of a particular software distribution. If the program is extracted from that 
%distribution and used or distributed within the terms of the program's 
%license, all parties to whom the program is redistributed should have the same 
%rights as those that are granted in conjunction with the original software 
%distribution.

%No item 9, "outros programas" não é a mesma coisa que outras partes de um 
%programa ao qual o programa original, modificado ou não, foi anexado. A idéia 
%é impedir que a licença de um programa que é distribuído "empacotado" com 
%outros influencie as licenças desses outros. Isso poderia ser usado para impor 
%restrições de uso de forma colateral. É importante enfatizar que "outros programas"
%são OUTRAS OBRAS, cuja distribuição é independente. Se esse item se aplicasse 
%à mesma obra, ele proibiria licenças de copyleft, o que claramente não é a intenção.
%The license must not place restrictions on other software that is distributed 
%along  with the licensed software. For example, the license must not insist 
%that all other programs distributed on the same medium must be open-source 
%software.

%\item[10] A licença deve ser neutra, em se tratando de tecnologia
% Justificativa retirada do sítio da Open Source Initiative: This provision is 
%aimed specifically at licenses which require an explicit gesture of assent in 
%order to establish a contract between licensor and licensee. Provisions 
%mandating so-called "click-wrap" may conflict with important methods of 
%software distribution such as FTP download, CD-ROM anthologies, and web 
%mirroring; such provisions may also hinder code re-use. Conformant licenses 
%must allow for the possibility that (a) redistribution of the software will 
%take place over non-Web channels that do not support click-wrapping of the 
%download, and that (b) the covered code (or re-used portions of covered code) 
%may run in a non-GUI environment that cannot support popup dialogues.
    \end{itemize}}
\end{frame}
