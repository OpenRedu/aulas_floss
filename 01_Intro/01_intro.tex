	\frametitle{Documentação e suporte a línguas têm a ver com \tyellow{tornar a aplicação acessível}}

\textit{{\bf Presentation and packaging} comprise a wide range of tasks, all revolving around the theme of \textbf{\tred{reducing the barrier to entry}}. Making the project \tred{inviting to the uninitiated} means writing user and developer documentation [...]. Many programmers unfortunately \bred{treat this work as being of secondary importance} to the code itself. [...] it's crucial that they (presentation and packaging) be a priority from the very start of the project.}

\begin{frame}
\frametitle{As Apresentações do Projeto}
O que deve ser mostrado em cada apresentação:\vspace{0.3cm}
\begin{enumerate}
\item Build funcionando e discussão sobre tecnologias envolvidas
  \begin{itemize}
  \item Deve mostrar que as tecnologias não são um entrave severo.
  \item Traduções nesta apresentação ou na próxima.
  \end{itemize}
  \vspace{0.2cm}
\item Estrutura geral do sistema e como ele pode ser modificado.
  \begin{itemize}
  \item Apresentar um patch implementado!
  \end{itemize}
  \vspace{0.2cm}
\item Primeiras contribuições
  \vspace{0.2cm}
\item Apresentação final
\end{enumerate}
